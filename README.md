# 🌍 Vex: AI-Powered Outbreak Detection for Africa

**Detect disease outbreaks in real-time using Google Vertex AI and geospatial mapping.**

## 🔥 Features
- **AI Outbreak Prediction**: LSTM model trained on health data.
- **Geospatial Heatmaps**: PostGIS + Google Maps integration.
- **Real-Time Alerts**: Firebase notifications for health workers.

## 🛠️ Tech Stack
- **Backend**: FastAPI (Python)
- **AI/ML**: Google Vertex AI (AutoML/LSTM)
- **Database**: PostgreSQL + PostGIS
- **Maps**: Google Maps API

## 🚀 Quick Start

### Prerequisites
- Python 3.10+
- Google Cloud Account (for Vertex AI)
- PostgreSQL with PostGIS

### Installation
```bash
git clone https://github.com/your-username/vex-ai-outbreak-detection.git
cd vex-ai-outbreak-detection
pip install -r backend/requirements.txt
